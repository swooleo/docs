快速开始

Swoole可以通过Pecl或者源码来进行安装。

Pecl（推荐）

首先通过pecl工具下载和编译Swoole扩展：

# 安装最新版本
pecl install swoole
# 安装指定版本
pecl install swoole-4.4.8

编译成功之后，把Swoole添加到php.ini配置文件里面。配置文件的路径可以通过如下命令进行查找，例如：

php --ini
Configuration File (php.ini) Path: /etc
Loaded Configuration File:         /etc/php.ini
Scan for additional .ini files in: /etc/php.d

推荐在additional .ini目录里面单独为Swoole创建一个配置文件，例如：

touch /etc/php.d/swoole.ini
cat > /etc/php.d/swoole.ini <<EOF
; Enable swoole extension module
extension=swoole.so
EOF

源码安装

下载一份已经release的Swoole源码：

curl -L https://github.com/swoole/swoole-src/archive/v4.4.12.tar.gz -o swoole-src-4.4.12.tar.gz \
&& tar -xvf swoole-src-4.4.12.tar.gz \
&& cd swoole-src-4.4.12 \
&& phpize \
&& ./configure \
&& make -j4 \
&& make install

然后按照上面的方式把Swoole添加到配置文件php.ini里面即可。

验证是否安装成功

php --ri swoole
